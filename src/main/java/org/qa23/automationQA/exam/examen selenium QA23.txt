#. Запустить браузер
#. Открыть url http://shop.demoqa.com/shop/
#. Перейти в меню и открыть список Default sorting 
#. В появившемся окне выбрать пункт Sort By Popularity 
#. После загрузки страницы кликнуть по категории Shoes (проверка что все товары из этой категории)*
#. Кликнуть по товару с названием "ROSE GOLD SIMPLE STRAP PLATFORM SANDALS" 
#. После загрузки информации о товаре проверить его заголовок(h1).
#. Вернуться на страницу со списком товаров. 
#. В поисковой строке "Search" ввести запрос "Sun Glasses" (проверка что все товары из этой категории)*
#. После загрузки страницы перейти к товару "QUAY AUSTRALIA X JLO REINA WHITE SUNGLASSES" и кликнуть по заголовку(h1)
#. После загрузки информации о товаре проверить цену акционного товара (₹55.00 вместо ₹60.00)
